O que é Arduino?

Arduino é uma plataforma que permite programar e controlar circuitos eletrônicos.

Principais conexões:

  • GND
  • VCC
  • LED
  • Digital
  • Analógico

Projeto: Semáforo Inteligente com Sensor Infravermelho

Ideia do projeto

A proposta foi criar um semáforo automático utilizando um sensor infravermelho para

detectar a presença de pessoas que querem pessar. A ideia simula um sistema real de

trânsito, onde o fluxo pode ser controlado de forma mais inteligente, sem depender

apenas de tempo fixo.

Como o projeto foi feito

O projeto foi montado utilizando uma placa Arduino conectada a uma protoboard. Os

LEDs representam as luzes do semáforo (vermelho, amarelo e verde), enquanto o

sensor infravermelho funciona como “os olhos” do sistema.

Materiais utilizados

  • 1 Arduino Uno
  • 1 Protoboard
  • 3 LEDs (vermelho, amarelo e verde)
  • 3 Resistores (geralmente 220Ω)
  • 1 Sensor infravermelho (IR)
  • 9 Jumpers (fios de conexão)
Aprendizados do projeto

Esse projeto foi como treinar um pequeno “cérebro eletrônico”. Durante o processo,

foi possível aprender:

  • Como funciona um circuito simples com LEDs
  • Uso de sensores na prática
  • Lógica de programação aplicada à eletrônica
  • Noções de automação e sistemas inteligentes
Conclusão

O semáforo com infravermelho mostrou como a robótica pode simular situações reais

de forma simples e eficiente. Mesmo com poucos componentes, foi possível criar um

sistema funcional e interativo.
